# Pliki dźwiękowe dla gry Milionerzy

Ten folder powinien zawierać następujące pliki dźwiękowe:

1. **intro.mp3** - Dźwięk czołówki odtwarzany przy rozpoczęciu gry
2. **new-question.mp3** - Dźwięk odtwarzany gdy pojawia się nowe pytanie
3. **correct-answer.mp3** - Dźwięk odtwarzany gdy odpowiedź jest poprawna
4. **wrong-answer.mp3** - Dźwięk odtwarzany gdy odpowiedź jest błędna
5. **audience-vote.mp3** - Dźwięk odtwarzany gdy aktywowane jest głosowanie publiczności

Pliki powinny być w formacie MP3 i umieszczone w tym folderze.
